Prioridad en Órdenes de Trabajo

Sincroniza la estrella de prioridad con la orden de producción

Este módulo agrega una estrella ⭐ de prioridad en las órdenes de trabajo de fabricación, visible en la vista lista. La prioridad se hereda automáticamente de la orden de producción. Útil para que los operarios puedan identificar órdenes prioritarias fácilmente sin modificar nada.

  • 📌 Compatible con Odoo 17
  • 📌 Totalmente integrado con MRP
  • 📌 Sincronización automática
  • 📌 Icono personalizado
